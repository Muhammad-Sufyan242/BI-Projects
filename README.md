# 🚖 Uber Data Analysis Dashboard (Power BI)

## 📌 Overview  
This project focuses on analyzing Uber ride data using Power BI to extract meaningful insights and build an interactive dashboard. It demonstrates how raw data can be transformed into clear, visual insights for better decision-making.

---

## 🎯 Objective  
The main objective of this project is to:  
- Analyze Uber ride data  
- Identify trends and patterns  
- Create an interactive dashboard  
- Present insights in a visual format  

---

## 🛠️ Tool Used  
- **Power BI** → Data cleaning, modeling, and visualization  

---

## 📊 Project Workflow  

### 1. Data Import  
- Loaded Uber dataset into Power BI  

### 2. Data Cleaning (Power Query)  
- Removed null values  
- Changed data types  
- Cleaned and structured data  

### 3. Data Modeling  
- Created relationships between multiple tables 
- Optimized data model  

### 4. Data Visualization  
- Built interactive dashboard  
- Used charts, KPIs, and slicers  

---

## 📈 Key Insights  
- Peak ride hours analysis  
- Most active locations  
- Ride trends over time  
- Demand patterns  

---

## 📌 Dashboard Features  
- Interactive filters (date, location)  
- KPI cards (total rides, etc.)  
- Clean and user-friendly visuals  
- Dynamic charts  

---

## 🚀 Project Outcome  
This dashboard helps in understanding ride patterns and supports data-driven decision-making using Power BI.
